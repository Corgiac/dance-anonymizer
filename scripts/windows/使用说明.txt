Windows 使用说明

前提：已安装 Python 3.10+
      安装时务必勾选 "Add Python to PATH"
      下载地址：https://www.python.org/downloads/

第一步：安装
  ① 双击 setup.bat （自动装依赖+合并模型）
  ② 看到 "Setup complete!" 后按任意键关闭

第二步：启动
  双击 run.bat
  看到 "Uvicorn running on..." 说明启动成功

第三步：打开网页
  浏览器输入 http://localhost:8002

第四步：手机也能用
  手机和电脑连同一个 WiFi
  手机浏览器输入 http://电脑IP:8002
  (电脑 IP：Win+R → cmd → ipconfig → 找 IPv4 地址)

========================================
注意：使用时建议关闭梯子/VPN，否则可能
      导致手机访问失败或下载速度异常
========================================
