## 改动 3: 降级形态学 — 删除 OPEN，改为自适应 CLOSE

**文件**: `src/effects.py` — `_morphology_clean()`

**删除**:
```python
def _morphology_clean(mask: np.ndarray) -> np.ndarray:
    mask_u8 = ...
    # Step1: CLOSE(3×3, iters=1)
    kernel_close = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (3, 3))
    closed = cv2.morphologyEx(mask_u8, cv2.MORPH_CLOSE, kernel_close, iterations=1)
    # Step2: OPEN(2×2)  ← 此行删除
    kernel_open = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (2, 2))
    opened = cv2.morphologyEx(closed, cv2.MORPH_OPEN, kernel_open)
    return opened > 0
```

**替换为**:
```python
def _morphology_clean(mask: np.ndarray) -> np.ndarray:
    """自适应形态学: 仅闭运算, 不做开运算避免腐蚀手臂/手指。
    小面积mask(远人物)用更小核, 大面积mask用正常核。"""
    mask_u8 = ...
    area = np.count_nonzero(mask_u8)
    if area < 5000:
        kernel_close = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (2, 2))
    else:
        kernel_close = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (3, 3))
    closed = cv2.morphologyEx(mask_u8, cv2.MORPH_CLOSE, kernel_close, iterations=1)
    return closed > 0
```

**变化**: OPEN(2×2) 删除 → 不再腐蚀手臂/手指/细肢；新增自适应核: area<5000→2×2, area≥5000→3×3
