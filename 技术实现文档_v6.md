# 舞蹈视频智能打码系统 — v6 Soft Alpha Pipeline

> 模型: yolo11s-seg + BoT-SORT | 全链路 float32 alpha | 2026-05-29

---

## 1. 核心架构变更 (v5 → v6)

**v5 问题**: 整个 pipeline 过早二值化 (`mask_bool = prob > 0.15`)，后续 morphology/contour/threshold 全部基于 bool mask，导致边缘抽搐、手臂消失、EMA 失效、多人合并。

**v6 方案**: 全链路 soft alpha (float32 [0,1])。从 YOLO 输出到最终渲染，不做任何 `> threshold` 二值化。

```
v5: prob → bool(>0.15) → morphology(bool) → contour(bool) → EMA(bool→float) → threshold(>0.45) → render

v6: prob → alpha(clip(prob-0.1)/0.4) → GaussianBlur → EMA(float) → GaussianBlur → render
```

---

## 2. 数据流

```
视频 → VideoReader.frames()
  │
  ▼
DanceTracker.track(frame)
  ├─ model.track(persist=True, tracker="botsort.yaml")
  ├─ prob → alpha = clip((prob-0.1)/0.4, 0, 1)     ← soft alpha 转换
  ├─ alpha.sum() < 800 → skip                        ← 小区域过滤
  ├─ BBox 膨胀 10%/5%
  └─ 时序 fallback: 未检测到 → cached_alpha * 0.85   ← 防闪断
  │
  ▼  TrackResult.mask = float32 alpha [0,1]
  │
  ▼
process_frame_effects(frame, track_results)
  ├─ _smooth_alpha(): GaussianBlur(5×5, σ=1.0)
  ├─ TemporalMaskCache.update(): EMA decay=0.7
  ├─ get_blended_body(): GaussianBlur(5×5, σ=1.0)
  ├─ get_blended_edge(): GaussianBlur(alpha) - alpha   ← soft edge
  ├─ calculate_depth_order(): EMA 80/20 + 30px死区
  └─ render: alpha compositing (solid/gradient/blur)
```

### 3.2 DanceTracker

```python
class DanceTracker:
    def __init__(self, config):
        self.model = YOLO(config.model_path)
        self._device = config.device
        self._fallback_alphas: dict = {}   # ← v6 时序 fallback 缓存

    def track(self, frame) -> List[TrackResult]:
        return self._detect_and_track(frame)

    def reset(self):
        self._fallback_alphas.clear()
```

### 3.3 _detect_and_track() — BoT-SORT + Soft Alpha

```python
results = self.model.track(
    source=frame, conf=0.3, iou=0.55, classes=[0],
    retina_masks=True, imgsz=1280,
    persist=True, tracker="botsort.yaml",
)

for each detection:
    # ★ Soft alpha 转换 (替代 bool 二值化)
    m = result.masks.data[i].cpu().numpy()
    alpha = np.clip((m - 0.1) / 0.4, 0.0, 1.0).astype(np.float32)

    # 过滤极小区域
    if alpha.sum() < self.config.min_alpha_sum: continue

    # BBox 膨胀 10%/5%
    # foot_y: alpha 行权重加权

    self._fallback_alphas[track_id] = alpha.copy()

# ★ 时序 fallback
for tid, cached in self._fallback_alphas.items():
    if tid not in seen_ids:
        decayed = cached * 0.85
        if decayed.sum() > 400:
            track_results.append(decayed)      # 补一帧
            self._fallback_alphas[tid] = decayed
        else:
            del self._fallback_alphas[tid]     # 彻底丢失
```

**alpha 转换公式**: `clip((prob - 0.1) / 0.4, 0, 1)`

| prob | alpha |
|------|-------|
| 0.10 | 0.00 |
| 0.20 | 0.25 |
| 0.30 | 0.50 |
| 0.50 | 1.00 |

保留了 0.1~0.5 之间的渐变过渡，边缘不再是硬切。

**时序 fallback**: BoT-SORT 某帧漏检时，用上一帧 alpha × 0.85 补位。sum > 400 才输出，< 400 删除缓存。防止突然消失。

## 4. src/effects.py

### 4.1 Alpha 平滑

```python
def _smooth_alpha(alpha: np.ndarray) -> np.ndarray:
    """GaussianBlur 替代 morphology, 平滑不伤细肢"""
    return np.clip(cv2.GaussianBlur(alpha, (5,5), sigmaX=1.0), 0, 1)
```

### 4.2 Soft Edge 提取

```python
def _extract_edge_alpha(alpha: np.ndarray, edge_width=3) -> np.ndarray:
    """GaussianBlur(alpha) - alpha = soft edge (替代 dilate-bool-subtract)"""
    expanded = cv2.GaussianBlur(alpha, (edge_width*2+1, edge_width*2+1), sigmaX=float(edge_width))
    edge = np.clip(expanded - alpha, 0, 1)
    return cv2.GaussianBlur(edge, (3,3), sigmaX=0.5)
```

### 4.3 EMA 时序缓存

```python
class TemporalMaskCache:
    def __init__(self, window_size=5, ema_decay=0.7):
        # EMA: new = 0.7*old + 0.3*current
        self._cache: Dict[int, np.ndarray] = {}

    def update(self, track_id, alpha, frame_idx):
        if track_id not in self._cache:
            self._cache[track_id] = alpha.copy()
        else:
            self._cache[track_id] = 0.7 * old + 0.3 * alpha

    def get_blended_body(self, track_id):
        alpha = self._cache.get(track_id)
        if alpha is None: return None
        return cv2.GaussianBlur(alpha, (5,5), sigmaX=1.0)

    def get_blended_edge(self, track_id, edge_width=3):
        alpha = self._cache.get(track_id)
        if alpha is None: return None
        smoothed = cv2.GaussianBlur(alpha, (5,5), sigmaX=1.0)
        return _extract_edge_alpha(smoothed, edge_width)
```

| 参数 | 值 | 位置 |
|------|-----|------|
| Alpha 平滑 | GaussianBlur(5,5) σ=1.0 | effects._smooth_alpha |

| Soft Edge | GaussianBlur(α) - α | _extract_edge_alpha |

### 7.4 已删除的 v5 操作

| 删除项 | 原因 |
|--------|------|
| `mask_bool = prob > 0.15` | 过早二值化 |
| `scipy.label` 噪点过滤 | 仅作用于 bool |
| `morphologyEx(MORPH_CLOSE)` | 腐蚀细肢 |
| `morphologyEx(MORPH_OPEN)` | 腐蚀细肢 |
| `findContours(RETR_EXTERNAL)` | 合并多人 |
| `drawContours(FILLED)` | 基于 bool |
| `blended_body > 0.45` | hard threshold 抽搐 |
| `exclusive_masks` | 互斥切割 |
| `occlusion_acc` | 遮挡累加 |
| `_bbox_to_mask` fallback | 矩形框伪造 |
| 自研 IoU greedy matcher | BoT-SORT 替代 |
