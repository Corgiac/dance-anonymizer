## 2. 整体数据流

```
视频文件 → VideoReader.frames() 逐帧 yied
    │
    ▼
DanceTracker.track(frame)
    ├─ _detect()         YOLO predict → 二值化 mask(>0.15) → 噪点过滤 → BBox 膨胀
    ├─ _associate()      IoU 贪心匹配 → lost 缓冲 → 恢复 → 创建
    ├─ _build_results()  active_tracks + ghost_tracks (≤1帧)
    └─ _impute_masks     直接透传 (不做修补)
    │
    ▼
process_frame_effects(frame, track_results)
    ├─ calculate_depth_order()  EMA 平滑 foot_y + 30px 死区
    ├─ 两阶段互斥切割:  近→远 (先占像素) → 远→近 (独占掩码渲染)
    ├─ _morphology_clean()      CLOSE(3×3,iters=1) + OPEN(2×2)
    ├─ _remove_internal_holes() 连通域最大轮廓填充
    ├─ _prepare_body_mask()     float32 转换 + 高斯模糊(3×3)
    ├─ TemporalMaskCache        EMA 关闭 (decay=0), 直接透传
    └─ apply_shadow_outline_effect()
          ├─ solid: alpha 混合填充色
          ├─ gradient: 竖向渐变 + mask 裁切
          ├─ blur: 高斯模糊 + mask 替换
          └─ border: dilate(3) - body → GaussianBlur → alpha 混合
    │
    ▼
VideoWriter → moviepy H.264 + AAC → 成品
```

### 3.4 track() 方法

```python
def track(self, frame):
    h, w = frame.shape[:2]
    detections = self._detect(frame)
    self._associate(detections, frame_shape=(h,w))
    self._purge_expired()
    results = self._build_results(frame_shape=(h,w))
    results = self._impute_masks(results, frame_shape=(h,w))
    self._frame_idx += 1
    return results
```

### 3.5 _detect() — YOLO 推理

```python
results = self.model.predict(
    source=frame,
    conf=0.3,
    iou=0.55,                     # 放宽 NMS
    classes=[0],                  # 仅 person
    retina_masks=True,            # 原生分辨率 mask
    imgsz=1280,                   # 推理分辨率
    device=self._device,
    verbose=False,
    half=self.config.half,
)
```

对每个检测结果:
1. 提取 bbox xyxy, confidence
2. `mask_bool = prob > 0.15` (低阈值捕获手臂)
3. 如果 `result.masks` 为 None → `continue` (直接跳过, 不用 BBox 伪造)
4. scipy.label 过滤 <100px 孤立噪点
5. 过滤 `np.count_nonzero < 1200`
6. BBox 非对称膨胀: 水平 10% × 垂直 5%

```python
bw, bh = x2 - x1, y2 - y1
expand_w = int(bw * 0.1)
expand_h = int(bh * 0.05)
x1 = max(0, x1 - expand_w)
y1 = max(0, y1 - expand_h)
x2 = min(w, x2 + expand_w)
y2 = min(h, y2 + expand_h)
```

### 3.6 _associate() — IoU 贪心匹配

```
1. 第一帧: 所有检测创建轨迹, stable_count = len(detections)
2. 构建 IoU 矩阵 (active_tracks × detections)
3. 贪心匹配: 按 IoU 降序, >= 0.35 关联
4. 未匹配 track: 前 3 帧留 active (复用历史 mask), 超过 3 帧才进 _lost_tracks
5. 未匹配 det: 尝试恢复 lost_track (IoU >= 0.35)
6. 剩余 det: active < stable_count → 创建新轨迹
7. lost_frames > 60 → 删除
```

### 3.7 _build_results()

```python
# active_tracks 全部输出
# lost_tracks 中 lost_frames <= ghost_frames(1) 作为幽灵帧输出
# 幽灵帧置信度衰减: conf * 0.9^lost_frames
```

## 4. src/effects.py — 特效渲染

### 4.1 形态学清洁

```python
def _morphology_clean(mask: np.ndarray) -> np.ndarray:
    # Step1: CLOSE(3×3, iters=1) 椭圆核 → 填补小孔洞
    kernel_close = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (3, 3))
    closed = cv2.morphologyEx(mask_u8, cv2.MORPH_CLOSE, kernel_close, iterations=1)
    # Step2: OPEN(2×2) 椭圆核 → 去边缘毛刺
    kernel_open = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (2, 2))
    opened = cv2.morphologyEx(closed, cv2.MORPH_OPEN, kernel_open)
    return opened > 0
```

### 4.2 连通域去孔洞

```python
def _remove_internal_holes(mask: np.ndarray) -> np.ndarray:
    contours, _ = cv2.findContours(mask_u8, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    if not contours: return mask
    max_contour = max(contours, key=cv2.contourArea)
    cv2.drawContours(filled, [max_contour], -1, 255, thickness=cv2.FILLED)
    return filled > 0
```

### 4.4 边缘提取

```python
def _extract_edge_mask(body_float, dilate_kernel=3, blur_sigma=1.0):
    body_u8 = (body_float > 0.4) * 255
    kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (3, 3))
    dilated = cv2.dilate(body_u8, kernel)
    edge_raw = dilated - body_u8
    blur_ks = dilate_kernel * 2 + 1  # = 7
    edge_float = edge_raw.astype(np.float32) / 255.0
    edge_blurred = cv2.GaussianBlur(edge_float, (7, 7), sigmaX=1.0)
    return edge_blurred  # float32 [0,1] 灰度抗锯齿
```

### 4.5 TemporalMaskCache (EMA 关闭版)

```python
class TemporalMaskCache:
    def __init__(self, window_size=5, ema_decay=0.0):
        self.ema_decay = ema_decay
        self._cache: Dict[int, np.ndarray] = {}

    def update(self, track_id, body_float, frame_idx):
        if track_id not in self._cache:
            self._cache[track_id] = body_float.copy()
        else:
            self._cache[track_id] = (
                self.ema_decay * self._cache[track_id]
                + (1.0 - self.ema_decay) * body_float
            )
        # ema_decay=0 时: cache = current → 完全不融合

    def get_blended_body(self, track_id):
        return self._cache.get(track_id)

    def get_blended_edge(self, track_id, dilate_kernel=3, blur_sigma=1.0):
        body = self.get_blended_body(track_id)
        return _extract_edge_mask(body, dilate_kernel, blur_sigma) if body is not None else None
```

### 4.6 深度排序 (EMA + 30px 死区)

```python
def calculate_depth_order(track_results, temporal_window=5, smooth_history=None):
    # smooth_history: {track_id: {"ema_y": float, "prev_y": float}}
    DEADZONE = 30

    # Step1: EMA 更新 foot_y
    for tr in track_results:
        if tid not in smooth_history:
            smooth_history[tid] = {"ema_y": curr_y, "prev_y": curr_y}
        else:
            entry["prev_y"] = entry["ema_y"]
            entry["ema_y"] = 0.8 * entry["ema_y"] + 0.2 * curr_y  # EMA 80/20

    # Step2: 按 EMA foot_y 升序 → [远, ..., 近]
    sorted_pairs = sorted(current_pairs, key=foot_y)
    ordered_ids = [tid for tid, _ in sorted_pairs]

    # Step3: 30px 死区 — 脚差 < 30px 时保持上一帧图层顺序不变
    if prev_order and same_ids:

    return ordered_ids, smooth_history
```

### 4.7 两阶段互斥切割 (apply_shadow_outline_effect)

```python
exclusive_masks = {}

for tid in reversed(depth_order):  # 近→远

# Step B: 远→近 用独占掩码渲染
for tid in depth_order:  # 远→近
    mask_bool = exclusive_masks[tid]
    if not mask_bool.any(): continue
```

## 7. api.py — FastAPI Web 服务

### 7.1 接口

| 接口 | 方法 | 功能 |
|------|------|------|
| `/` | GET | 三步交互前端 |
| `/analyze` | POST | 上传→扫描12帧选最佳→默认打码→返回效果图+IDs |
| `/preview_frame` | POST | 单帧重渲染 (~50ms, 滑块实时反馈) |
| `/preview_snippet` | POST | 3秒片段 (async, 可中断) |
| `/render` | POST | 全片渲染 (async, threading.Event 取消) |
| `/cleanup/{id}` | DELETE | 释放服务端资源 |

### 7.2 前端交互

- **Step1**: 上传视频 → /analyze → 返回打码效果图 + 人物 ID 列表
- **Step2**: 勾选人物 + 拖滑块调参 → 300ms 防抖 → /preview_frame 实时刷新
- **按钮互斥**: 点击 [3秒预览] → 另一按钮置灰 → 变红色 [取消] → 再点 abort()
- **Step3**: 渲染完成 → 视频播放器 + 下载按钮
- **重新上传**: resetToStep1() — abort + DELETE cleanup + 数据归零 + UI 复位

### 7.3 关键帧选择

```
扫描 12 帧 (均匀分布) → YOLO predict 快速 count → 取人数最多帧
最佳帧上跑完整 tracker → 预览图渲染 + 标注 ID (黑底白字)
```
