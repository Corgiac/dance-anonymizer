## 2. 数据流

```
视频文件
  │
  ▼ VideoReader.frames() — 逐帧 yield
  │
  ├─→ DanceTracker.track(frame)
  │     ├─ _detect()      → YOLO predict → 二值化 mask (>0.25) → 过滤小目标
  │     ├─ _associate()   → IoU 贪心匹配 → lost | 恢复 | 新建
  │     ├─ _build_results → active_tracks + ghost_tracks (lost≤3帧)
  │     └─ _impute_masks  → 直接透传 (不做修补)
  │
  ▼ process_frame_effects(frame, track_results)
  │     ├─ calculate_depth_order() → 脚部Y坐标排序
  │     ├─ morphology_close(3×3)   → 填补破洞
  │     ├─ body float32 转换
  │     ├─ TemporalMaskCache       → 直接透传当前帧 (EMA 关闭)
  │     └─ apply_shadow_outline_effect()
  │           ├─ 填充: solid / gradient / blur
  │           └─ 边框: dilate(3) - body → GaussianBlur → alpha 混合
  │
  ▼ VideoWriter → 临时 mp4v → moviepy 转 H.264 + AAC → 成品
```

## 3. src/tracker.py — 分割 & 追踪

### 3.1 配置参数

```python
@dataclass
class TrackerConfig:
    model_path: str = "yolo11s-seg.pt"
    device: str = "cuda"
    conf_threshold: float = 0.3
    iou_threshold: float = 0.35
    track_buffer: int = 60
    iou_match_threshold: float = 0.3
    expected_count: Optional[int] = None
    half: bool = False
    verbose: bool = True
    ghost_frames: int = 3
    min_mask_area: int = 2000
```

### 3.2 检测 _detect()

```
model.predict(
    source=frame, conf=0.3, iou=0.35, classes=[0],
    retina_masks=True, imgsz=1024, device=device,
    verbose=False, half=half
)
```

每个检测:
- 提取 bbox (xyxy), confidence, mask
- `mask_bool = prob > 0.25` — 低阈值捕获手臂等低置信度区域
- `resize_mask()` 如尺寸不匹配 → INTER_NEAREST
- 过滤: `np.count_nonzero(mask) < 2000` → 跳过 (防杯子误检)

### 3.3 追踪 _associate()

```
1. 第一帧: 所有检测创建轨迹 → 确定 stable_count
2. IoU 矩阵 (active_tracks × detections)
3. 贪心匹配: 按 IoU 降序, >= 0.3 关联
4. 未匹配 active → _lost_tracks, lost_frames += 1
5. 未匹配 det + 未匹配 lost: 最高 IoU >= 0.3 → 恢复
6. 剩余 det + active < stable_count → 创建新轨迹
7. lost_frames > 60 → 删除
```

### 3.4 输出 _build_results()

```
active_tracks 全部输出
+ lost_tracks 中 lost_frames <= 3 (幽灵帧, 复用历史 mask, 置信度衰减 ×0.9^n)
```

### 3.5 _impute_masks()

```
直接透传 return results  (不做任何跨帧修补)
```

## 4. src/effects.py — 特效渲染

### 4.1 形态学清洁

```python
# MORPH_CLOSE (3×3) — 先膨胀后腐蚀, 填补内部破洞, 不伤边缘
cv2.morphologyEx(mask_u8, cv2.MORPH_CLOSE, kernel=3×3 ones)
```

### 4.2 身体 mask 准备

```python
# 直接 float32 转换, body_expand_pixels=0 (贴骨, 不膨胀)
body_float = opened_mask.astype(np.float32) / 255.0
```

### 4.3 边缘提取

```python
# dilate(ELLIPSE 3×3) - body → GaussianBlur(7×7, σ=1.0) → float32 [0,1]
body_u8 = (body_float > 0.4) * 255
dilated = cv2.dilate(body_u8, ELLIPSE(3,3))
edge_raw = dilated - body_u8
edge_blurred = cv2.GaussianBlur(edge_raw/255.0, (7,7), σ=1.0)
```

### 4.4 TemporalMaskCache

```
EMA 关闭 (decay=1.0): get_blended_body() 直接返回 history[-1] (当前帧)
不做任何跨帧融合。
```

### 4.5 填充渲染

| 模式 | 实现 |
|------|------|
| solid | `result = src*(1-α) + fill_color*α` (alpha 混合, 支持 opacity) |
| gradient | 竖向线性渐变 (top→bot) × mask 裁切 |
| blur | GaussianBlur(31px) × mask 区域替换 |

### 4.6 边框渲染

```
result = src*(1-edge_α) + border_color*edge_α
edge_α 来自高斯模糊后的灰度图, 天然抗锯齿
```

### 4.7 深度排序

```
foot_y = mask 中最大行索引 (脚部)
4 帧移动平均平滑 foot_y
按 foot_y 升序渲染: 远→近 (远者先画, 近者后画覆盖)
```

## 5. api.py — FastAPI Web 服务

### 5.1 接口

| 接口 | 方法 | 功能 |
|------|------|------|
| `/` | GET | Web UI |
| `/analyze` | POST | 上传→扫描12帧选最佳→默认打码→返回效果图+IDs |
| `/preview_frame` | POST | 单帧重渲染 (~50ms) |
| `/preview_snippet` | POST | 3秒片段 (async, 可中断) |
| `/render` | POST | 全片渲染 (async, threading.Event 取消) |
| `/cleanup/{id}` | DELETE | 释放服务端资源 |

### 5.2 关键帧选择

```
扫描 12 帧均匀分布 → YOLO predict 快速 count → 取人数最多的帧
对最佳帧跑完整 tracker 获取 mask + track_id
预览图渲染后标注 ID 标签 (黑底白字)
```

### 5.3 互斥与取消

```
前端: AbortController + 按钮互斥 (点击A→B置灰→A变红"取消")
后端: request.is_disconnected() 检测 → break 循环
      pipeline 通过 threading.Event 检查取消信号
```
