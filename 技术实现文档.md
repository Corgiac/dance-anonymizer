## 1. 系统概述

### 1.1 项目定位

面向固定机位舞蹈视频（3-8人）的智能打码/特效渲染工具。自动检测人体，分配稳定ID，按脚部深度排序，渲染"纯黑阴影 + 抗锯齿白色轮廓"特效，输出音画同步的成品视频。

### 1.2 核心能力

| 能力 | 实现方式 |
|------|---------|
| 人体分割 | YOLOv8s-seg + `retina_masks=True` + `imgsz=1024`，输出原生分辨率高精度mask |
| 多目标追踪 | 自研 IoU 贪心匹配追踪器，`track_buffer=60` 遮挡恢复 |
| 深度排序 | 脚部Y坐标法 (Y大→离相机近→后渲染) + 4帧移动平均防抖动 |
| 阴影渲染 | EMA时序平滑 body_mask → alpha混合 → 软边缘无闪烁黑影 |
| 白边渲染 | 贴骨边缘提取 → 3px椭圆核 → 高斯模糊抗锯齿 → EMA时序融合 |
| 音画同步 | moviepy 从原视频提取音频流 → 无损叠加 |

### 1.3 整体数据流

```
输入视频 (任意格式)
    │
    ▼
VideoReader ──逐帧(yield)──→ DanceTracker.track()
                                  │
                    YOLO predict(retina_masks, imgsz=1024)
                                  │
                    IoU 贪心匹配 (track_buffer=60)
                                  │
                    List[TrackResult] (id, bbox, mask, foot_y)
                                  │
                                  ▼
                    process_frame_effects()
                      ├─ morphology_open(5×5)
                      ├─ prepare_body_mask (贴骨, 0膨胀)
                      ├─ TemporalMaskCache.update (EMA)
                      ├─ 阴影: blended_body alpha混合
                      └─ 白边: blended_edge alpha混合
                                  │
                                  ▼
VideoWriter ──逐帧写入──→ 临时视频 (无音频)
                                  │
                    moviepy 提取原音频
                                  │
                                  ▼
                            最终成品.mp4
```

### 3.1 VideoReader — 完整源码

```python
import os
import cv2
import numpy as np
from typing import Iterator, Optional, Tuple


class VideoReader:
    """
    逐帧视频读取器，支持上下文管理器。

    使用方式:
        with VideoReader("input.mp4") as reader:
            for frame in reader.frames():
                process(frame)
    """

    def __init__(self, video_path: str):
        self.video_path = video_path
        if not os.path.exists(video_path):
            raise FileNotFoundError(f"视频文件不存在: {video_path}")

        self._cap: Optional[cv2.VideoCapture] = None
        self._fps: float = 0.0
        self._total_frames: int = 0
        self._width: int = 0
        self._height: int = 0
        self._fourcc: str = ""

    def __enter__(self) -> "VideoReader":
        self._cap = cv2.VideoCapture(self.video_path)
        if not self._cap.isOpened():
            raise IOError(f"无法打开视频文件: {self.video_path}")

        self._fps = self._cap.get(cv2.CAP_PROP_FPS)
        self._total_frames = int(self._cap.get(cv2.CAP_PROP_FRAME_COUNT))
        self._width = int(self._cap.get(cv2.CAP_PROP_FRAME_WIDTH))
        self._height = int(self._cap.get(cv2.CAP_PROP_FRAME_HEIGHT))
        fourcc_int = int(self._cap.get(cv2.CAP_PROP_FOURCC))
        self._fourcc = "".join([chr((fourcc_int >> 8 * i) & 0xFF) for i in range(4)])
        return self

    def __exit__(self, *args):
        if self._cap is not None:
            self._cap.release()

    @property
    def fps(self) -> float:
        return self._fps

    @property
    def total_frames(self) -> int:
        return self._total_frames

    @property
    def width(self) -> int:
        return self._width

    @property
    def height(self) -> int:
        return self._height

    @property
    def frame_size(self) -> Tuple[int, int]:
        return (self._width, self._height)

    @property
    def fourcc(self) -> str:
        return self._fourcc

    def frames(self) -> Iterator[Tuple[int, np.ndarray]]:
        """逐帧迭代器，返回 (帧索引, BGR图像)。内存优化: 不累积帧数据。"""
        idx = 0
        while True:
            ret, frame = self._cap.read()
            if not ret:
                break
            yield idx, frame
            idx += 1

    def get_frame(self, index: int) -> Optional[np.ndarray]:
        """随机访问指定帧 (非高性能，调试用)。"""
        self._cap.set(cv2.CAP_PROP_POS_FRAMES, index)
        ret, frame = self._cap.read()
        return frame if ret else None
```

```python
class VideoWriter:
    """
    逐帧视频写出器，支持上下文管理器。
    使用方式:
        with VideoWriter("output.mp4", fps=30, size=(1920, 1080)) as writer:
            for frame in processed_frames:
                writer.write(frame)
    """

    def __init__(self, video_path: str, fps: float, size: Tuple[int, int],
                 fourcc: str = "mp4v"):
        self.video_path = video_path
        self.fps = fps
        self.size = size
        self.fourcc = fourcc
        os.makedirs(os.path.dirname(video_path) or ".", exist_ok=True)
        self._writer: Optional[cv2.VideoWriter] = None

    def __enter__(self) -> "VideoWriter":
        fourcc_int = cv2.VideoWriter_fourcc(*self.fourcc)
        self._writer = cv2.VideoWriter(
            self.video_path, fourcc_int, self.fps, self.size
        )
        if not self._writer.isOpened():
            raise IOError(f"无法创建输出视频: {self.video_path}")
        return self

    def __exit__(self, *args):
        if self._writer is not None:
            self._writer.release()

    def write(self, frame: np.ndarray):
        """写入单帧。frame 必须为 BGR uint8 且尺寸匹配构造函数。"""
        self._writer.write(frame)
```

```python
def has_audio_stream(video_path: str) -> bool:
    """检测视频是否包含音频流 (使用 moviepy)。"""
    try:
        from moviepy.video.io.VideoFileClip import VideoFileClip
        clip = VideoFileClip(video_path)
        has_audio = clip.audio is not None
        clip.close()
        return has_audio
    except Exception:
        return False


def merge_audio_with_moviepy(
    video_path: str,
    audio_source_path: str,
    output_path: str
) -> str:
    """
    使用 moviepy 将无声视频与原始音频合成为最终成品。
    保证帧率和时长与原视频严格一致。
    """
    from moviepy.video.io.VideoFileClip import VideoFileClip
    from moviepy.audio.io.AudioFileClip import AudioFileClip

    video_clip = VideoFileClip(video_path)
    audio_clip = AudioFileClip(audio_source_path)

    # 对齐时长: 取较短的
    duration = min(video_clip.duration, audio_clip.duration)
    video_clip = video_clip.subclipped(0, duration)
    audio_clip = audio_clip.subclipped(0, duration)

    final = video_clip.with_audio(audio_clip)
    final.write_videofile(
        output_path,
        codec="libx264",
        audio_codec="aac",
        logger=None,
    )
    video_clip.close()
    audio_clip.close()
    final.close()
    return output_path


def get_video_info(video_path: str) -> dict:

            "width": reader.width,

            "fps": reader.fps,
```

```python
    """单个人物的追踪结果"""
    track_id: int                           # 全局唯一人物ID (从0开始自增)

    mask: np.ndarray                        # (H, W) bool 分割掩码
```

### 4.2 DanceTracker 类 — 完整源码

```python
class DanceTracker:
    """
    舞蹈视频人物追踪器
    架构: YOLOv8-Seg predict + 自研 IoU 追踪器
    专为固定机位舞蹈场景优化:
      - predict 模式避免 ultralytics track() 的兼容性问题
      - track_buffer 实现遮挡后 ID 恢复
      - 固定相机: 人数稳定后抑制新目标
      - 全局ID固化: 同一人物全程ID唯一

    使用方式:
        tracker = DanceTracker(TrackerConfig())
        for frame in video_reader.frames():
            tracks = tracker.track(frame)
    """

    def __init__(self, config: TrackerConfig = TrackerConfig()):
        self.config = config
        if config.verbose:
            print(f"[DanceTracker] 加载模型: {config.model_path}")
        self.model = YOLO(config.model_path)
        self._device = config.device

        # ---- 追踪器内部状态 ----
        self._next_id: int = 0                        # 下一个可分配的 track_id
        self._active_tracks: dict = {}                # {track_id: _Tracklet}
        self._lost_tracks: dict = {}                  # {track_id: _Tracklet} 丢失缓冲
        self._frame_idx: int = 0
        self._stable_count: Optional[int] = None      # 稳定后的人数
```

### 4.3 `track()` — 公开接口源码

```python
    def track(self, frame: np.ndarray) -> List[TrackResult]:
        """对单帧执行分割+追踪。"""
        h, w = frame.shape[:2]

        # Step 1: YOLOv8-Seg 预测 (仅 detect person)
        detections = self._detect(frame)

        # Step 2: 关联检测结果到已有轨迹
        self._associate(detections, frame_shape=(h, w))

        # Step 3: 回收丢失超时的轨迹
        self._purge_expired()

        # Step 4: 生成当前帧结果
        results = self._build_results(frame_shape=(h, w))

        self._frame_idx += 1
        return results

    def reset(self):
        """重置追踪器状态 (处理新视频时调用)。"""
        self._next_id = 0
        self._active_tracks.clear()
        self._lost_tracks.clear()
        self._frame_idx = 0
        self._stable_count = None
```

### 4.4 `_detect()` — YOLO 推理源码

```python
    def _detect(self, frame: np.ndarray) -> List[dict]:
        """
        YOLOv8-Seg 预测，返回检测列表。
        每个检测: {"bbox": (x1,y1,x2,y2), "conf": float, "mask": (H,W) bool}
        """
        results = self.model.predict(
            source=frame,
            conf=self.config.conf_threshold,     # 默认 0.3
            iou=self.config.iou_threshold,       # 默认 0.35
            device=self._device,
            classes=[0],                         # 仅检测 person (COCO class 0)
            verbose=False,
            half=self.config.half,
            retina_masks=True,                   # ★ 高精度原生分辨率 mask
            imgsz=1024,                          # ★ 推理分辨率 1024
        )

        detections = []
        result = results[0]
        if result is None or result.boxes is None or len(result.boxes) == 0:
            return detections

        h, w = frame.shape[:2]

        for i in range(len(result.boxes)):
            bbox_xyxy = result.boxes.xyxy[i].cpu().numpy()
            x1, y1, x2, y2 = bbox_xyxy.astype(int).tolist()
            conf = float(result.boxes.conf[i].item())

            # mask 提取与 resize
            if result.masks is not None:
                m = result.masks.data[i].cpu().numpy()
                mask_bool = m > 0.5
                if mask_bool.shape != (h, w):
                    mask_bool = cv2_resize_mask(mask_bool, w, h)
            else:
                mask_bool = _bbox_to_mask((x1, y1, x2, y2), h, w)

            detections.append({
                "bbox": (x1, y1, x2, y2),
                "conf": conf,
                "mask": mask_bool,
            })

        return detections
```

**参数说明：**
- `classes=[0]`: 仅检测 COCO 第 0 类 (person)，过滤掉其他 79 个类别
- `retina_masks=True`: 输出与原图同分辨率的 mask，消除低分上采样的锯齿和精度损失
- `imgsz=1024`: 推理分辨率提升至 1024×1024，确保细长手臂等小目标不被降采样吞没
- `conf=0.3`: 降低置信度阈值以捕获四肢等低置信度区域

### 4.5 `_associate()` — IoU 贪心匹配源码

```python
    def _associate(self, detections: List[dict], frame_shape: tuple):
        """
        将当前帧检测结果与已有轨迹进行 IoU 匹配。
        匹配策略 (简化 Hungarian):
          1. 计算所有 active_track × detection 的 IoU 矩阵
          2. 贪心匹配: 每次取最高 IoU 配对，高于阈值则关联
          3. 未匹配的检测 → 可能创建新轨迹 (需满足稳定条件)
          4. 未匹配的轨迹 → 标记为 lost，进入缓冲期
        """
        active_ids = list(self._active_tracks.keys())
        if not active_ids and not detections:
            return

        # 初始化: 第一个有人帧，直接创建轨迹
        if not active_ids and detections:
            if self._stable_count is None:
                for d in detections:
                    self._create_tracklet(d, frame_shape)
                if self.config.expected_count:
                    self._stable_count = self.config.expected_count
                else:
                    self._stable_count = len(detections)
            return

        # 构建 IoU 矩阵
        n_tracks = len(active_ids)
        n_dets = len(detections)
        iou_mat = np.zeros((n_tracks, n_dets), dtype=np.float32)
        for i, tid in enumerate(active_ids):
            t_bbox = self._active_tracks[tid].last_bbox
            for j, d in enumerate(detections):
                iou_mat[i, j] = _box_iou(t_bbox, d["bbox"])

        matched_track = set()
        matched_det = set()
        pairs = []

        # 贪心匹配: 按 IoU 降序
        flat_indices = np.argsort(iou_mat.ravel())[::-1]
        for idx in flat_indices:
            i = idx // n_dets
            j = idx % n_dets
            if i in matched_track or j in matched_det:
                continue
            if iou_mat[i, j] >= self.config.iou_match_threshold:
                pairs.append((i, j))
                matched_track.add(i)
                matched_det.add(j)

        # ---- 执行匹配 ----
        for i, j in pairs:
            tid = active_ids[i]
            det = detections[j]
            tracklet = self._active_tracks[tid]
            tracklet.update(det)

        # ---- 处理未匹配的轨迹 (进入 lost 缓冲) ----
        for i, tid in enumerate(active_ids):
            if i not in matched_track:
                tracklet = self._active_tracks.pop(tid)
                tracklet.lost_frames += 1
                self._lost_tracks[tid] = tracklet

        # ---- 尝试恢复 lost 轨迹 (对未匹配的检测) ----
        lost_ids = list(self._lost_tracks.keys())
        dets_still_unmatched = [j for j in range(n_dets) if j not in matched_det]

        for lost_tid in lost_ids:
            lt = self._lost_tracks[lost_tid]
            best_j = -1
            best_iou = self.config.iou_match_threshold
            for j in dets_still_unmatched:
                iou = _box_iou(lt.last_bbox, detections[j]["bbox"])
                if iou > best_iou:
                    best_iou = iou
                    best_j = j

            if best_j >= 0:
                lt.update(detections[best_j])
                lt.lost_frames = 0
                self._active_tracks[lost_tid] = lt
                del self._lost_tracks[lost_tid]
                dets_still_unmatched.remove(best_j)

        # ---- 处理仍不匹配的检测 (创建新轨迹) ----
        # 仅在稳定人数未满时允许新增
        current_total = len(self._active_tracks)
        for j in dets_still_unmatched:
            if self._stable_count is not None and current_total >= self._stable_count:
                break  # 已达稳定人数，忽略新目标
            self._create_tracklet(detections[j], frame_shape)
            current_total += 1
```

### 4.6 IoU 计算与辅助函数 — 完整源码

```python
def _box_iou(box1, box2):
    """计算两个 bbox 的 IoU (均为 xyxy 格式)。"""
    x_left = max(box1[0], box2[0])
    y_top = max(box1[1], box2[1])
    x_right = min(box1[2], box2[2])
    y_bottom = min(box1[3], box2[3])

    if x_right <= x_left or y_bottom <= y_top:
        return 0.0
    inter = (x_right - x_left) * (y_bottom - y_top)
    area1 = (box1[2] - box1[0]) * (box1[3] - box1[1])
    area2 = (box2[2] - box2[0]) * (box2[3] - box2[1])
    return inter / (area1 + area2 - inter + 1e-6)


class _Tracklet:
    """单个跟踪轨迹。"""
    __slots__ = ("tid", "last_bbox", "last_conf", "last_mask",
                 "lost_frames", "created_frame", "bbox_history")

    def __init__(self, tid: int, first_det: dict, frame_idx: int,
                 frame_shape: tuple):
        self.tid = tid
        self.last_bbox = first_det["bbox"]
        self.last_conf = first_det["conf"]
        self.last_mask = first_det["mask"]
        self.lost_frames = 0
        self.created_frame = frame_idx
        self.bbox_history = deque(maxlen=5)
        self.bbox_history.append(first_det["bbox"])

    def update(self, det: dict):
        """用新检测更新轨迹。"""
        self.last_bbox = det["bbox"]
        self.last_conf = det["conf"]
        self.last_mask = det["mask"]
        self.lost_frames = 0
        self.bbox_history.append(det["bbox"])


def cv2_resize_mask(mask: np.ndarray, w: int, h: int) -> np.ndarray:
    """将 mask resize 到 (h, w)，使用最近邻插值保持二值性。"""
    import cv2
    return cv2.resize(
        mask.astype(np.uint8), (w, h),
        interpolation=cv2.INTER_NEAREST
    ).astype(bool)


def _bbox_to_mask(bbox: Tuple[int, int, int, int], h: int, w: int) -> np.ndarray:
    """将 bbox 转为矩形 mask (无分割时的 fallback)。"""
    mask = np.zeros((h, w), dtype=bool)
    x1, y1, x2, y2 = bbox
    x1c, y1c = max(0, x1), max(0, y1)
    x2c, y2c = min(w, x2), min(h, y2)
    if x2c > x1c and y2c > y1c:
        mask[y1c:y2c, x1c:x2c] = True
    return mask
```

```
  │

  └── 白边渲染: blended_edge → alpha 混合 [255,255,255]
```

### 5.2 优化1: 形态学开运算 — 完整源码

```python
def _morphology_open(mask: np.ndarray, kernel_size: int = 5) -> np.ndarray:
    """
    优化1: 形态学开运算 (Morphological Opening)
    先腐蚀再膨胀, 消除离群的小碎块像素和微小孔洞。
    参数:
        mask:        (H, W) bool 原生YOLO二值化mask
        kernel_size: 开运算核尺寸 (奇数)
    返回:
        (H, W) uint8 清洁后的mask (0/255)
    """
    mask_u8 = mask.astype(np.uint8) * 255
    kernel = np.ones((kernel_size, kernel_size), np.uint8)
    opened = cv2.morphologyEx(mask_u8, cv2.MORPH_OPEN, kernel, iterations=1)
    return opened
```

### 5.3 优化2: 身体 mask 准备 (贴骨级) — 完整源码

```python
def _prepare_body_mask(opened_mask: np.ndarray,
                        expand_pixels: int = 0) -> np.ndarray:
    """
    优化2: 身体mask准备 (贴骨级精度)
    依赖 retina_masks + imgsz=1024 的高精度YOLO输出,
    不再做全局膨胀补偿。仅将 uint8 mask 转为 float32 供时序缓存使用。
    参数:
        opened_mask:   (H, W) uint8 开运算后的mask (0/255)
        expand_pixels: 膨胀像素数 (默认0=贴骨, >=2才做极小膨胀)
    返回:
        (H, W) float32 身体mask [0.0, 1.0]
    """
    body_float = opened_mask.astype(np.float32) / 255.0

    if expand_pixels >= 2:
        ksize = expand_pixels * 2 + 1
        kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (ksize, ksize))
        expanded = cv2.dilate(opened_mask, kernel, iterations=1)
        body_float = expanded.astype(np.float32) / 255.0
        body_float = cv2.GaussianBlur(body_float, (3, 3), sigmaX=0.5)

    return np.clip(body_float, 0.0, 1.0)
```

### 5.4 优化3: 轮廓提取 + 抗锯齿 — 完整源码

```python
def _extract_edge_mask(
    body_mask_float: np.ndarray,
    dilate_kernel_size: int = 3,   # ★ 纤细贴骨白边
    blur_sigma: float = 1.0,
) -> np.ndarray:
    """
    优化3: 从身体mask提取边缘 + 高斯模糊抗锯齿
    步骤:
      1. 将body_mask_float二值化(>0.5)回到uint8
      2. dilate - body = 边缘环
      3. GaussianBlur → 灰度过渡
    返回:
        (H, W) float32 灰度边缘mask [0.0, 1.0]
    """
    # Step 1: 二值化
    body_u8 = (body_mask_float > 0.4).astype(np.uint8) * 255

    if not body_u8.any():
        return np.zeros_like(body_mask_float, dtype=np.float32)

    # Step 2: 椭圆核膨胀 → 边缘环 = dilated - body
    kernel = cv2.getStructuringElement(
        cv2.MORPH_ELLIPSE,
        (dilate_kernel_size, dilate_kernel_size)
    )
    dilated = cv2.dilate(body_u8, kernel, iterations=1)
    edge_raw = cv2.subtract(dilated, body_u8)

    if not edge_raw.any():
        return np.zeros_like(body_mask_float, dtype=np.float32)

    # Step 3: 高斯模糊 → 灰度过渡抗锯齿
    blur_ks = dilate_kernel_size * 2 + 1   # 3*2+1 = 7
    if blur_ks % 2 == 0:
        blur_ks += 1

    edge_float = edge_raw.astype(np.float32) / 255.0
    edge_blurred = cv2.GaussianBlur(edge_float, (blur_ks, blur_ks),
                                     sigmaX=blur_sigma)
    return np.clip(edge_blurred, 0.0, 1.0)
```

**关键**: `MORPH_ELLIPSE` 椭圆核使白边保持人体轮廓的自然弧度。高斯模糊后的 edge_mask 是 float32 灰度图 `[0, 1]`，渲染时作为 alpha 通道进行软混合。

### 5.5 优化4: 时序 EMA 缓存 `TemporalMaskCache` — 完整源码

这是系统**防闪烁的核心资产**。

```python
class TemporalMaskCache:
    """
    优化4: 时序抗闪烁滤波器 (v2 — 身体+边缘共用缓存)
    缓存每 track_id 最近 N 帧的 body_mask_float,
    阴影和轮廓共用同一份时序平滑数据。

    参数:
        window_size: 时序窗口大小 (默认8帧)
        ema_decay:   EMA衰减系数 (0=只用最新, 1=完全平均, 默认0.5)
    """

    def __init__(self, window_size: int = 8, ema_decay: float = 0.5):
        self.window_size = window_size
        self.ema_decay = ema_decay
        self._cache: Dict[int, deque] = {}       # {track_id: deque([float32_mask × N])}
        self._ema: Dict[int, np.ndarray] = {}    # {track_id: EMA累积器}
        self._last_seen: Dict[int, int] = {}

    def update(self, track_id: int, body_mask_float: np.ndarray, frame_idx: int):
        """
        存入当前帧的body_mask到缓存并更新EMA。
        """
        if track_id not in self._cache:
            self._cache[track_id] = deque(maxlen=self.window_size)
        self._cache[track_id].append(body_mask_float.copy())
        self._last_seen[track_id] = frame_idx

        # EMA 更新: ema_new = decay × ema_old + (1-decay) × current
        if track_id not in self._ema:
            self._ema[track_id] = body_mask_float.copy()
        else:
            self._ema[track_id] = (
                self.ema_decay * self._ema[track_id]
                + (1.0 - self.ema_decay) * body_mask_float
            )

    def get_blended_body(self, track_id: int) -> Optional[np.ndarray]:
        """
        获取时序平滑后的身体mask (EMA优先, 冷启动时用简单平均)。
        返回: (H, W) float32 [0.0, 1.0] 或 None
        """
        ema = self._ema.get(track_id)
        if ema is not None:
            return ema.copy()
        # 冷启动fallback: 简单平均
        history = self._cache.get(track_id)
        if not history:
            return None
        stacked = np.stack(list(history), axis=0)
        return np.mean(stacked, axis=0)

    def get_blended_edge(self, track_id: int,
                          dilate_kernel_size: int = 3,
                          blur_sigma: float = 1.0) -> Optional[np.ndarray]:
        """
        从时序平滑后的身体mask提取抗锯齿边缘。
        返回: (H, W) float32 [0.0, 1.0] 或 None
        """
        body = self.get_blended_body(track_id)
        if body is None:
            return None
        return _extract_edge_mask(body, dilate_kernel_size, blur_sigma)

    def cleanup_stale(self, active_ids: set):
        """清理不活跃的track_id缓存 (释放内存)。"""
        for tid in list(self._cache.keys()):
            if tid not in active_ids:
                del self._cache[tid]
                self._ema.pop(tid, None)
                self._last_seen.pop(tid, None)

    def reset(self):
        """清空全部缓存。"""
        self._cache.clear()
        self._ema.clear()
        self._last_seen.clear()
```

**EMA 公式** (实际使用 `decay=0.4`):

```
EMA_t = 0.4 × EMA_{t-1} + 0.6 × X_t

权重分配:
  当前帧 t:  权重 0.6
  帧 t-1:    权重 0.6 × 0.4 = 0.24
  帧 t-2:    权重 0.6 × 0.4² = 0.096
  帧 t-3:    权重 0.6 × 0.4³ = 0.038
  ...
  总和趋近于 1.0
```

**关键设计**: 阴影和白边**共用同一份时序缓存**。阴影从 `get_blended_body()` 渲染，白边从 `get_blended_edge()` 渲染，边缘天然带有和 body 相同的时序平滑。

### 5.6 深度排序 — 完整源码

```python
def calculate_depth_order(
    track_results: List,
    temporal_window: int = 5,
    smooth_history: Optional[dict] = None,
) -> Tuple[List[int], dict]:
    """
    基于脚部Y坐标计算深度排序 (远→近)。
    规则:
      - foot_y (mask最低有效像素的Y坐标) 越大 → 离相机越近 → 后渲染 (画在上层)
      - 排序结果: [最远的人, ..., 最近的人]，按此顺序渲染
    时域平滑: 对 foot_y 应用 N 帧移动平均，抑制逐帧抖动导致的层级跳变
    """
    if smooth_history is None:
        smooth_history = {}

    for tr in track_results:
        tid = tr.track_id
        if tid not in smooth_history:
            smooth_history[tid] = deque(maxlen=temporal_window)
        smooth_history[tid].append(tr.foot_y)

    smoothed_pairs = []
    for tr in track_results:
        tid = tr.track_id
        history = smooth_history.get(tid, deque([tr.foot_y]))
        smooth_y = np.mean(history) if history else tr.foot_y
        smoothed_pairs.append((tid, smooth_y))

    sorted_pairs = sorted(smoothed_pairs, key=lambda p: p[1])
    ordered_ids = [tid for tid, _ in sorted_pairs]

    return ordered_ids, smooth_history
```

### 5.7 渲染核心 `apply_shadow_outline_effect()` — 完整源码

```python
def apply_shadow_outline_effect(
    frame: np.ndarray,
    depth_order: List[int],
    track_id_to_idx: dict,
    track_results: List,
    temporal_cache: TemporalMaskCache,
    frame_idx: int,
    open_kernel_size: int = 5,
    body_expand_pixels: int = 0,       # ★ 贴骨级
    dilate_kernel_size: int = 3,       # ★ 纤细白边
    blur_sigma: float = 1.0,
) -> np.ndarray:
    """
    对帧应用阴影 + 抗锯齿白色轮廓特效 (v2 — 身体+边缘均时序平滑)。
    渲染管线:
      优化1: morphology_open(original_mask) → 清洁
      优化2: prepare_body_mask → float32转换 (不膨胀, 贴骨)
      优化3: 存入时序缓存 → EMA加权 → 提取边缘
      优化4: 用平滑后的body_mask渲染阴影 + 平滑后的edge_mask渲染白边
    """
    h, w = frame.shape[:2]
    result = frame.copy()

    # ---- 逐层渲染 (远→近) ----
    for tid in depth_order:
        idx = track_id_to_idx.get(tid)
        if idx is None:
            continue

        tr = track_results[idx]
        mask_bool = tr.mask

        if not mask_bool.any():
            continue

        # ============================================================
        # 优化1: 形态学开运算
        # ============================================================
        opened_u8 = _morphology_open(mask_bool, kernel_size=open_kernel_size)

        # ============================================================
        # 优化2: 身体mask准备 (贴骨级, 依赖 retina_masks 高精度输出)
        # ============================================================
        body_float = _prepare_body_mask(opened_u8, expand_pixels=body_expand_pixels)

        # ============================================================
        # 优化3+4: 存入时序缓存 → 获取平滑后的body和edge
        # ============================================================
        temporal_cache.update(tid, body_float, frame_idx)

        blended_body = temporal_cache.get_blended_body(tid)
        if blended_body is None:
            blended_body = body_float

        blended_edge = temporal_cache.get_blended_edge(
            tid, dilate_kernel_size=dilate_kernel_size, blur_sigma=blur_sigma
        )
        if blended_edge is None:
            blended_edge = _extract_edge_mask(
                blended_body, dilate_kernel_size, blur_sigma
            )

        # ============================================================
        # 阴影渲染: 用时序平滑后的body mask
        # ============================================================
        body_valid = blended_body > 0.1
        if body_valid.any():
            # 核心身体区域(>0.5): 直接填黑
            core = blended_body > 0.5
            result[core] = [0, 0, 0]
            # 边缘过渡区域(0.1~0.5): alpha混合
            edge_zone = body_valid & (~core)
            if edge_zone.any():
                a_body = blended_body[edge_zone, np.newaxis]
                result[edge_zone] = (
                    result[edge_zone].astype(np.float32) * (1.0 - a_body)
                ).astype(np.uint8)

        # ============================================================
        # 白边渲染: 用时序平滑后的edge mask
        # ============================================================
        outline_valid = blended_edge > 0.005
        if outline_valid.any():
            a_edge = blended_edge[outline_valid, np.newaxis]
            result[outline_valid] = (
                result[outline_valid].astype(np.float32) * (1.0 - a_edge)
                + 255.0 * a_edge
            ).astype(np.uint8)

    return result
```

**阴影双阈值设计：**

| 区域 | body_mask 值 | 渲染方式 | 效果 |
|------|-------------|---------|------|
| 核心身体 | > 0.5 | 直接填 `[0,0,0]` | 纯黑覆盖 |
| 边缘过渡 | 0.1 ~ 0.5 | `result × (1-α)` | 半透明渐变 |
| 外部 | < 0.1 | 不渲染 | 原帧保留 |

**白边混合公式**:
```
result = result × (1 - edge_alpha) + 255 × edge_alpha
  edge_alpha = 0.5 → 50%原色 + 50%白色 = 柔和白边
  edge_alpha = 0.9 → 10%原色 + 90%白色 = 明亮白边
```

### 5.8 `process_frame_effects()` — 完整源码

```python
def process_frame_effects(
    frame: np.ndarray,
    track_results: List,
    temporal_cache: Optional[TemporalMaskCache] = None,
    smooth_history: Optional[dict] = None,
    frame_idx: int = 0,
    open_kernel_size: int = 5,
    body_expand_pixels: int = 0,       # ★ 贴骨级: 0=不膨胀
    dilate_kernel_size: int = 3,       # ★ 纤细白边: 3
    blur_sigma: float = 1.0,
    temporal_window: int = 8,
    target_ids: Optional[List[int]] = None,
) -> Tuple[np.ndarray, dict, TemporalMaskCache]:
    """
    单帧完整特效处理 (v2)。
    返回: (result_frame, smooth_history, temporal_cache)
    """
    if temporal_cache is None:
        temporal_cache = TemporalMaskCache(
            window_size=temporal_window, ema_decay=0.4
        )

    if target_ids is not None:
        target_set = set(target_ids)
        track_results = [tr for tr in track_results if tr.track_id in target_set]

    if not track_results:
        return frame.copy(), (smooth_history or {}), temporal_cache

    # 深度排序
    ordered_ids, smooth_history = calculate_depth_order(
        track_results,
        temporal_window=max(3, temporal_window // 2),  # = 4
        smooth_history=smooth_history,
    )

    # 建立 track_id → index 映射
    track_id_to_idx = {tr.track_id: i for i, tr in enumerate(track_results)}

    # 应用特效
    result = apply_shadow_outline_effect(
        frame=frame,
        depth_order=ordered_ids,
        track_id_to_idx=track_id_to_idx,
        track_results=track_results,
        temporal_cache=temporal_cache,
        frame_idx=frame_idx,
        open_kernel_size=open_kernel_size,
        body_expand_pixels=body_expand_pixels,
        dilate_kernel_size=dilate_kernel_size,
        blur_sigma=blur_sigma,
    )

    # 清理过期缓存
    active_ids = {tr.track_id for tr in track_results}
    temporal_cache.cleanup_stale(active_ids)

    return result, smooth_history, temporal_cache
```

```python
        target_ids: Optional[List[int]] = None,

    ) -> str:

                  f"@ {info['fps']:.2f}fps, {info['total_frames']} 帧")

        temporal_cache: Optional[TemporalMaskCache] = None

        # 从配置读取特效参数
        open_kernel_size = self.effect_config.get("open_kernel_size", 5)
        body_expand_pixels = self.effect_config.get("body_expand_pixels", 0)
        dilate_kernel_size = self.effect_config.get("dilate_kernel_size", 3)
        blur_sigma = self.effect_config.get("blur_sigma", 1.0)
        temporal_window = self.effect_config.get("temporal_window", 8)

        tmp_video = output_path + ".tmp_video.mp4"

        try:
            with VideoReader(input_path) as reader:
                with VideoWriter(tmp_video, reader.fps, reader.frame_size) as writer:
                    pbar = tqdm(total=reader.total_frames, desc="处理中",
                                unit="帧", disable=not show_progress)
                    start_time = time.time()
                    processed_count = 0

                    for frame_idx, frame in reader.frames():
                        try:
                            # Step 1: 追踪
                            track_results = self._tracker.track(frame)

                            # Step 2: 特效渲染 (v2 — 身体+边缘均时序平滑)
                            result_frame, smooth_history, temporal_cache = \
                                process_frame_effects(
                                    frame=frame,
                                    track_results=track_results,
                                    temporal_cache=temporal_cache,
                                    smooth_history=smooth_history,
                                    frame_idx=frame_idx,
                                    open_kernel_size=open_kernel_size,
                                    body_expand_pixels=body_expand_pixels,
                                    dilate_kernel_size=dilate_kernel_size,
                                    blur_sigma=blur_sigma,
                                    temporal_window=temporal_window,
                                    target_ids=target_ids,
                                )

                            # Step 3: 写入
                            writer.write(result_frame)
                            processed_count += 1

                        except Exception as e:
                            # 异常降级: 保留原帧
                            if show_progress:
                                tqdm.write(
                                    f"  [警告] 第{frame_idx}帧处理异常: {e}, "
                                    f"已降级为原帧"
                                )
                            writer.write(frame)
                            processed_count += 1

                        pbar.update(1)

                    pbar.close()
                    elapsed = time.time() - start_time
                    if show_progress:
                        fps_processing = processed_count / elapsed if elapsed > 0 else 0
                        print(f"[Pipeline] 处理完成: {processed_count} 帧 "
                              f"耗时 {elapsed:.1f}s ({fps_processing:.2f} fps)")

            # ---- 音频合并 ----
            if show_progress:
                print("[Pipeline] 检查音频流...")

            if has_audio_stream(input_path):
                if show_progress:
                    print("[Pipeline] 合成音视频 (moviepy)...")
                merge_audio_with_moviepy(tmp_video, input_path, output_path)
                if show_progress:
                    print(f"[Pipeline] 最终成品: {output_path}")
            else:
                if os.path.exists(output_path):
                    os.remove(output_path)
                os.rename(tmp_video, output_path)
                if show_progress:
                    print(f"[Pipeline] 最终成品 (无音频): {output_path}")

        finally:
            # 清理临时视频
            if os.path.exists(tmp_video):
                try:
                    os.remove(tmp_video)
                except OSError:
                    pass

        return output_path

    def reset(self):
        """重置所有内部状态。"""
        if self._tracker:
            self._tracker.reset()
```

## 9. 核心算法详解

### 9.1 IoU 贪心匹配追踪

**时间复杂度**: O(N×M + K log K)，K = N×M。舞蹈场景 N、M 均 ≤ 8，实际计算量极小。

**算法步骤**:

```
输入: active_tracks[N], detections[M]

Step 1: 构建 IoU 矩阵 mat[N×M]
        mat[i][j] = box_iou(track_i.last_bbox, det_j.bbox)

Step 2: 将所有 (i,j) 对按 IoU 降序排序, 贪心选取:
        若 i,j 均未匹配 且 mat[i][j] >= 0.3:
          track_i.update(det_j), 标记已匹配

Step 3: 未匹配的 tracks → _lost_tracks, lost_frames += 1

Step 4: 未匹配的 dets 尝试恢复 lost_tracks:
        对每个 lost_track, 在未匹配 dets 中找最高 IoU >= 0.3 → 恢复

Step 5: 仍有未匹配 dets 且 active < stable_count → 创建新轨迹

Step 6: 清理 lost_tracks[lost_frames > 60]
```

### 9.2 EMA 时序平滑 (防闪烁核心)

**动机**: YOLO 每帧独立推理，mask 边界天然有 2-5px 随机抖动。直接用当前帧 mask 渲染，黑影轮廓每帧都在跳。

**公式**:

```
EMA_t = decay × EMA_{t-1} + (1-decay) × X_t

decay = 0.4:
  当前帧 t:   权重 0.6
  帧 t-1:     权重 0.6 × 0.4 = 0.24
  帧 t-2:     权重 0.6 × 0.4² = 0.096
  帧 t-3:     权重 0.6 × 0.4³ = 0.038
  ...

EMA 对最新帧响应更快 (权重 0.6 vs 简单平均的 1/N), 且仅需 O(WH) 内存。
```

### 9.3 脚部 Y 坐标深度排序

```
原理: 固定机位下, 脚部在画面中的Y坐标直接反映人物距相机的远近。
      Y越大 (越靠近画面底部) → 距离相机越近 → 渲染越靠后 (画在上层)。

foot_y = max(np.where(mask == True)[0])   # mask 中非零像素的最大行索引
smooth_foot_y = mean(最近4帧的 foot_y)     # 移动平均防抖动

渲染顺序: [最远的人, ..., 最近的人]  (Painter's Algorithm)
```

### 9.4 Alpha 混合渲染公式

**阴影（双阈值）**:
```
核心 (>0.5):  result = [0, 0, 0]                # 纯黑覆盖
边缘 (0.1~0.5): result = result × (1 - alpha)   # 半透明黑
外部 (<0.1):  不渲染                              # 原帧保留
```

**白边（灰度混合）**:
```
result = result × (1 - edge_alpha) + 255 × edge_alpha
```

### 9.5 形态学操作流程

```
原始 mask (bool)
    │ cv2.morphologyEx(src, MORPH_OPEN, 5×5 kernel)
    │   先腐蚀: 消除孤立碎块、毛刺
    │   再膨胀: 恢复主体区域
    ▼
清洁后的 mask (uint8, 0/255)
    │ _prepare_body_mask (expand_pixels=0, 贴骨模式)
    │   直接转 float32 / 255.0
    ▼
body_mask (float32, [0.0, 1.0])
    │ TemporalMaskCache EMA 平滑
    ├── 阴影渲染: blended_body > 0.5 → [0,0,0]
    └── _extract_edge_mask
         │ cv2.dilate(body, ELLIPSE(3,3)) - body = 边缘环
         │ cv2.GaussianBlur(edge, (7,7), sigma=1.0) → 灰度抗锯齿
         └── 白边渲染: alpha 混合 [255,255,255]
```

## 10. 性能与内存设计

| 保证项 | 实现方式 |
|--------|---------|
| 内存安全 | `VideoReader.frames()` 生成器逐帧 yield；处理后立即释放 |
| 时序缓存内存 | 每 track_id 存 8 帧 float32 mask (720p ≈ 8×2.8MB ≈ 22MB 上限) |
| 异常降级 | 单帧异常 → 保留原帧 + tqdm.write 警告 |
| 输出一致性 | `VideoWriter` 严格使用 `reader.fps` 和 `reader.frame_size` |
| 音画同步 | moviepy `subclipped(0, min(duration))` 对齐时长 |
| 临时文件清理 | `finally` 块保证 `os.remove(tmp_video)` |
| 编码兼容 | 输入 H.264/H.265，输出统一 H.264 (`mp4v` + `libx264`) |

**720p CPU 性能基准** (yolov8s-seg, imgsz=1024, retina_masks, 730帧):

| 阶段 | 耗时 | 占比 |
|------|------|------|
| YOLO predict | ~387s | ~85% |
| 形态学操作 + EMA + 渲染 | ~45s | ~10% |
| 视频 I/O + 音频合成 | ~23s | ~5% |
| **总计** | **~455s (1.6 fps)** | 100% |
